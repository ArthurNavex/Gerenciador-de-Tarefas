# Gerenciador-de-Tarefas
Gerenciador de tarefas para usuarios

Projeto criado para que eu pudesse treinar minha logica de programação e aprender sobre c#.

Funcionalidades: 
  *Cadastrar Usuarios
  *Ver Usuarios cadastrados
  *Adicionar tarefas para usuarios
  *Ver tarefas dos usuarios 
  *Marcar tarefas como concluidas

Como rodar o projeto:
  *É necessário ter o SDK do .Net 
  1 - No terminal, navegue até a pasta do projeto
  2 - Compile e execute
